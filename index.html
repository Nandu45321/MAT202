Discrete Distributions

Bernoulli Distribution
\[ P(X=0) = 1 - p \] \[ P(X=1) = p \]
\[ E[X] = p \] \[ \text{Var}[X] = p(1-p) \]
Models a single trial with two possible outcomes (success/failure). Parameter p is the probability of success.
Binomial Distribution
\[ P(X=i) = \binom{n}{i} p^i (1-p)^{n-i} \]
\[ E[X] = np \] \[ \text{Var}[X] = np(1-p) \]
Models the number of successes in n independent Bernoulli trials. Parameters: n (number of trials), p (success probability).
Geometric Distribution
\[ P(X=i) = (1-p)^{i-1} p \]
\[ E[X] = \frac{1}{p} \] \[ \text{Var}[X] = \frac{1-p}{p^2} \]
Models the number of trials needed to get the first success in a sequence of independent Bernoulli trials.
Poisson Distribution
\[ P(X=x) = \frac{e^{-\lambda} \lambda^x}{x!} \]
\[ E[X] = \lambda \] \[ \text{Var}[X] = \lambda \]
Models the number of events occurring in a fixed interval of time or space. Parameter λ is the average rate.

Continuous Distributions

Uniform Distribution
\[ f(x) = \frac{1}{b-a} \quad \text{for } a \leq x \leq b \]
\[ E[X] = \frac{a+b}{2} \] \[ \text{Var}[X] = \frac{(b-a)^2}{12} \]
All intervals of the same length have equal probability. Parameters: a (minimum), b (maximum).
Normal Distribution
\[ f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \]
\[ E[X] = \mu \] \[ \text{Var}[X] = \sigma^2 \]
The most common continuous distribution. Parameters: μ (mean), σ (standard deviation).
Standard Normal Distribution
\[ Z \sim N(0, 1) \] \[ \Phi(-z) = 1 - \Phi(z) \]
Normal distribution with mean 0 and variance 1. Φ(z) is the cumulative distribution function.
Exponential Distribution
\[ f(x) = \lambda e^{-\lambda x} \quad \text{for } x \geq 0 \]
\[ E[X] = \frac{1}{\lambda} \] \[ \text{Var}[X] = \frac{1}{\lambda^2} \]
Models the time between events in a Poisson process. Parameter λ is the rate parameter.

Special Distributions

Gamma Distribution
\[ f(x) = \frac{\lambda^{\alpha} x^{\alpha-1} e^{-\lambda x}}{\Gamma(\alpha)} \quad \text{for } x > 0 \]
\[ \Gamma(\alpha) = \int_0^{\infty} e^{-y} y^{\alpha-1} dy \] \[ \Gamma(n) = (n-1)! \quad \text{for integer } n \]
\[ E[X] = \frac{\alpha}{\lambda} \] \[ \text{Var}[X] = \frac{\alpha}{\lambda^2} \]
Generalization of the exponential distribution. Parameters: α (shape), λ (rate).
Beta Distribution
\[ f(x) = \frac{x^{\alpha-1} (1-x)^{\beta-1}}{B(\alpha, \beta)} \quad \text{for } 0 < x < 1 \]
\[ B(\alpha, \beta) = \int_0^1 x^{\alpha-1} (1-x)^{\beta-1} dx \]
\[ E[X] = \frac{\alpha}{\alpha + \beta} \] \[ \text{Var}[X] = \frac{\alpha \beta}{(\alpha + \beta)^2 (\alpha + \beta + 1)} \]
Models random variables limited to intervals of finite length. Parameters: α, β > 0.
Linear Transformations
\[ E[aX + b] = aE[X] + b \] \[ \text{Var}[aX + b] = a^2 \text{Var}[X] \]
Formulas for expectation and variance of linearly transformed random variables.
Moment Generating Function (Normal)
\[ M_X(t) = e^{\mu t + \frac{1}{2} \sigma^2 t^2} \]
The moment generating function for a normal distribution.
